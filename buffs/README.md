# ⚔️ Buff-System

Das **Buff-System** ist das Herzstück von BuffSMP. Jeder Spieler besitzt **genau einen Buff** — eine einzigartige Fähigkeit, die deinen Spielstil definiert.

---

## 🧭 Wo fange ich an?

<!-- tabs:start -->

#### **🆕 Ich bin neu**

1. Lies die [Systemübersicht](system-overview.md) — Grundmechaniken, Punkte, Shop.
2. Schau dir den [Buff Guide](buff-guide.md) an — alle verfügbaren Buffs im Überblick.
3. Check die [Buff Tags](buff-tags.md) — welcher Spielstil passt zu dir?
4. Verstehe das [Altar-Ritual](altar-ritual.md) — wie du Buffs aktivierst.

#### **🎯 Ich kenne mich aus**

- [Essence Guide](essence-guide.md) — Wie du deinen Buff massiv verstärkst.
- [Rezepte](recipes.md) — Crafting-Rezepte aller Buff-Items.
- [Systemübersicht](system-overview.md) — Punkte, Shop, Commands.
